Data Cleaning Process
Checked the dataset for duplicate records using Show ID as the unique identifier. No duplicate records were found.
Confirmed and corrected data types for all columns.
Missing values in the Director column were replaced with "Unknown".
Missing values in the Cast and Country columns were replaced with "Not Specified".
Created a Year column using DAX for time-based analysis.
Key Insights
Movies dominate the Netflix catalog, accounting for approximately 70% of all titles, while TV Shows make up about 30%.
The United States is the leading content-producing country, contributing approximately 2.8K titles, significantly more than any other country.
Netflix experienced rapid content growth between 2016 and 2021, with the highest number of content additions recorded in 2019.
TV-MA is the most common content rating on Netflix, with approximately 3.2K titles.
Drama and International Movies are the most common genres on Netflix, followed by Documentaries and Stand-Up Comedy.
